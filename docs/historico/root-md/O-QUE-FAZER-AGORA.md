# O que fazer agora (sem Excel/CSV)

## Fluxo principal: **Ctrl+C → Ctrl+V**

1. Product Explorer → estrutura E-BOM visível.
2. Selecione **várias linhas** na grade → **Ctrl+C**.
3. Dashboard no Web Page Reader → caixa **Colar** → **Ctrl+V** → **Importar estrutura**.

URL:

```
https://mouraenderson.github.io/HTML-PRODUCT-EXPLORE/index.html
```

**Ctrl+F5** após atualizar o repositório.
