Objetivo: dashboard da E-BOM do Product Explorer

Este HTML foi feito para consumir a estrutura e atributos do Product Structure Explorer via APIs ENOVIA (mesma sessão 3DEXPERIENCE). Com o link GitHub no widget, só é possível visualizar layout ou dados de exemplo — não a BOM real do tenant.

Fallback manual (não substitui o Explorer)

Arraste o Physical Product do Explorer para a caixa — ou cole linhas da E-BOM

Arrastar produto ou colar E-BOM
  1. Produto: arraste o assembly (ex.: Drone) da árvore do Explorer para a caixa → Importar estrutura.
  2. E-BOM completa: selecione linhas na grade → Ctrl+C → Ctrl+V → Importar.

Vincular ao Product Explorer (aba EXPLORE)

Carregue o produto no Product Structure Explorer (aba EXPLORE), depois vincule aqui.

  1. Abra a aba EXPLORE e carregue o Physical Product (ex.: Drone).
  2. Volte à aba PRODUCTEXPLORE e clique Sincronizar.
  3. Se não atualizar: copie o Physical ID do Explorer e cole abaixo → Carregar objeto.
  4. Para BOM completa no HTML: coloque os dois widgets na mesma aba ou publique o HTML no 3DSpace.
