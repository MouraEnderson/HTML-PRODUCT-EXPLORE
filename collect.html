Coletor — Structure Explorer → JSON

Gera o arquivo que o BOM Analytics lê no 3DDashboard (sem deploy 3DSpace).

Passos
  1. No Product Structure Explorer, abra o assembly (ex. Mont10).
  2. Na grade E-BOM, selecione as linhas (Shift+clique ou Ctrl+A).
  3. Ctrl+C e cole na caixa abaixo.
  4. Clique Gerar JSON → baixe o arquivo.
  5. Envie o arquivo para quem mantém o GitHub ou cole direto no widget BOM.
